# Panduan Siswa

Selamat datang di Super Apps MATSANDATAMA — sistem akademik digital madrasah.

## 1. Login Pertama

Setelah login pertama dengan password default, sistem akan menyarankan Anda mengubah password. **Lakukan segera** demi keamanan akun.

Langkah ganti password:
1. Akan muncul popup setelah login — atau dari menu Profil → "Ubah Password".
2. Masukkan password lama.
3. Buat password baru (minimal 6 karakter, beda dari yang lama).
4. Konfirmasi password baru.
5. Klik **Simpan**.

## 2. Dashboard

Dashboard siswa menampilkan:
- **Pengumuman penting** dari sekolah (di paling atas)
- Jadwal pelajaran hari ini
- Akses cepat ke rapor & prestasi

Klik ikon 🔔 di kanan atas untuk melihat **semua notifikasi**.

## 3. Jadwal Saya

Menu **Jadwal Saya** menampilkan grid jadwal mingguan kelas Anda. Klik mata pelajaran tertentu untuk melihat detail (guru, ruang, materi terakhir).

## 4. Rapor Saya

Menu **Rapor Saya** menampilkan nilai semua mata pelajaran:
- Nilai Pengetahuan + Keterampilan + Nilai Akhir
- Predikat (A/B/C/D)
- Deskripsi per mata pelajaran
- Rata-rata semester

Anda dapat memilih semester untuk melihat rapor periode tertentu.

## 5. Data Prestasi

Upload prestasi Anda sendiri di menu **Data Prestasi**:
1. Klik **Tambah Prestasi**.
2. Isi:
   - Nama lomba / kompetisi
   - Bidang (akademik / olahraga / seni / keagamaan)
   - Tingkat (sekolah / kota / provinsi / nasional / internasional)
   - Peringkat (Juara 1/2/3/Harapan/Partisipan)
   - Penyelenggara
   - Tanggal pelaksanaan
3. Lampirkan foto sertifikat (opsional).
4. Submit — wali kelas akan memverifikasi.

Prestasi yang sudah **terverifikasi** akan masuk ke rekap sekolah & ditampilkan di dashboard kelas Anda.

## 6. Ekstrakurikuler

Menu **Ekstrakurikuler** menampilkan kegiatan ekskul yang Anda ikuti:
- Jadwal latihan
- Daftar hadir Anda
- Nilai akhir semester dari pembina

## 7. Pengumuman

Klik **Pengumuman** di sidebar atau di dashboard untuk melihat semua pengumuman sekolah:
- Info / Pengumuman umum
- Peringatan (deadline, kewajiban)
- Berita penting (libur, lomba, dll)

Pengumuman terbaru otomatis muncul di dashboard — jika sudah dibaca, akan kehilangan badge "BARU".

## Tips

- Buka aplikasi setiap pagi — ada pengumuman penting yang muncul di dashboard.
- **Jangan share password** Anda ke teman — setiap aktivitas tercatat di log audit.
- Klik ikon 🔔 di kanan atas untuk melihat semua notifikasi & pengumuman.
- Sesi akan otomatis logout setelah 30 menit tidak aktif — selalu logout manual di komputer umum.
- Jika lupa password, klik "Lupa Password" di halaman login (perlu email terdaftar).
